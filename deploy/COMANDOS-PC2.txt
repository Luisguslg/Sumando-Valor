PASOS (PC 2 - donde haces pull)
==============================

1) Ir a la carpeta del repo (raíz: donde está SumandoValor.sln)

cd "C:\RUTA\A\Sumando Valor APP"

2) Hacer pull

git pull

3) Generar ZIP listo para el servidor (Release)

powershell -ExecutionPolicy Bypass -File .\deploy\01_publish.ps1

4) Copiar al servidor (por RDP) estos 2 archivos:
   - artifacts\SumandoValor_publish.zip  ->  C:\Temp\SumandoValor_publish.zip  (en el servidor)
   - deploy\02_iis_install.ps1          ->  C:\Temp\02_iis_install.ps1        (en el servidor)
